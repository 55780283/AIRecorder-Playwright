# AI Recorder 悬浮侧边栏使用指南

## ✨ 新功能：悬浮侧边栏

现在 AI Recorder 支持**悬浮侧边栏**模式，可以在录制时固定在屏幕右侧，实时显示录制日志！

## 🎯 打开侧边栏的两种方式

### 方式 1：通过右键菜单（推荐）

```
1. 在任意网页上点击鼠标右键
2. 选择"打开 AI Recorder 侧边栏"
3. 侧边栏会悬浮在屏幕右侧
```

### 方式 2：通过 popup 按钮

```
1. 点击浏览器工具栏的 AI Recorder 图标
2. 点击"打开悬浮侧边栏"按钮
3. 侧边栏会自动打开
```

## 📱 侧边栏功能

### 1. 实时录制状态

- 🔴 **状态指示器**：顶部显示当前录制状态
- 📊 **操作计数**：显示已录制的操作步骤数
- 📝 **SOP 名称**：显示当前录制的 SOP 名称

### 2. 实时日志显示

- 🎨 **颜色编码**：不同操作类型有不同颜色
- ⏱️ **时间戳**：精确显示每个操作的时间
- 📜 **自动滚动**：始终显示最新的操作
- 🔄 **实时更新**：每 500ms 刷新一次

### 3. 控制按钮

- ⏺ **开始录制**：点击开始录制
- ⏹ **停止录制**：点击停止录制
- 🗑️ **清空日志**：清空当前日志显示

## 💡 使用场景

### 场景 1：边录制边查看日志

```
1. 右键打开侧边栏
2. 点击"开始录制"
3. 在页面中执行操作
4. 侧边栏实时显示录制的步骤
5. 确认所有操作都被记录
6. 点击"停止录制"
```

**优势**：
- ✅ 无需切换窗口查看日志
- ✅ 实时确认录制正常
- ✅ 发现问题及时停止

### 场景 2：长时间录制监控

```
1. 打开侧边栏开始录制
2. 侧边栏悬浮在右侧
3. 执行复杂的跨页面流程
4. 随时查看已录制的步骤
5. 确保流程完整
```

**优势**：
- ✅ 始终可见录制状态
- ✅ 了解录制进度
- ✅ 增强信心

### 场景 3：调试和排错

```
1. 打开侧边栏
2. 开始录制
3. 执行操作
4. 观察日志中的每个步骤
5. 检查时间间隔和顺序
```

**优势**：
- ✅ 直观看到每个操作
- ✅ 分析操作间隔
- ✅ 发现潜在问题

## ⚙️ 侧边栏特性

### 1. 悬浮固定

- 📍 **位置**：固定在屏幕右侧
- 🔄 **跟随**：切换标签页时跟随
- 💎 **透明**：背景半透明，不遮挡视线

### 2. 智能更新

- ⏱️ **刷新率**：500ms 自动更新
- 🎯 **按需更新**：只在有新操作时刷新
- 🚀 **性能优化**：最小化 DOM 操作

### 3. 状态保持

- 💾 **持久化**：关闭 popup 不影响录制
- 🔄 **自动恢复**：页面跳转后自动恢复
- 📊 **实时同步**：与 background 保持同步

## 🔧 技术实现

### 窗口管理

```javascript
// 创建悬浮窗口
await chrome.windows.create({
  url: chrome.runtime.getURL('sidebar/sidebar.html'),
  type: 'popup',
  width: 350,
  height: 600,
  left: screen.width - 350, // 屏幕右侧
  top: Math.max(0, (screen.height - 600) / 2),
  focused: true
});
```

### 右键菜单

```javascript
// 创建右键菜单项
chrome.contextMenus.create({
  id: 'open-sidebar',
  title: '打开 AI Recorder 侧边栏',
  contexts: ['all']
});
```

### 日志更新

```javascript
// 每 500ms 更新一次
setInterval(async () => {
  if (this.isRecording) {
    await this.updateRecordingLog();
  }
}, 500);
```

## 🐛 故障排除

### 问题 1：侧边栏打不开

**检查**：
1. 浏览器是否允许弹出窗口
2. 扩展是否有足够权限

**解决**：
- 允许 Chrome 弹出窗口
- 在 `chrome://extensions/` 重新加载扩展

### 问题 2：侧边栏日志不更新

**检查**：
1. 是否正在录制中
2. 是否执行了操作

**解决**：
- 确保点击了"开始录制"
- 在页面中执行一些操作
- 刷新侧边栏

### 问题 3：侧边栏位置不对

**解决**：
- 关闭侧边栏重新打开
- 调整屏幕分辨率后重新打开
